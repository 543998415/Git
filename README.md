#### Git和GitHub的使用

工作区->暂存区->本地分支master->远程分支master

期间 可以通过 git status 查看状态

1.添加

​	```git add ./A``` 

2.提交
​	```git commit -m '修改内容'```
3.推送
​	```git push```

4.github 通过pull request 提交代码到源仓库

如果 pull request没有通过。你下次push依旧会保存在该pullrequest中
